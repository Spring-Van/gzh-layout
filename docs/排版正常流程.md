# 微信公众号图文上传完整流程指南

## 一、前置条件

### 1. 公众号配置
- **认证要求**：需完成微信认证的订阅号或服务号
- **IP白名单**：在「设置与开发 → 基本配置 → IP白名单」中添加服务器IP
  - *错误示例*：`40164 invalid ip 175.152.205.147 ipv6 ::ffff:175.152.205.147, not in whitelist`
- **AppID & AppSecret**：在「基本配置」中获取，用于获取 AccessToken

### 2. 素材限制
| 类型 | 上限 | 说明 |
|------|------|------|
| 永久图片素材 | 10万个 | 包含封面图 |
| 永久图文素材 | 10万个 | 图文消息 |
| 临时素材 | 3天有效 | 不推荐用于封面 |

---

## 二、完整接口流程

### 流程概览
```
获取 AccessToken 
    ↓
上传封面图（永久素材）→ 获得 thumb_media_id
    ↓
上传正文图片（uploadimg）→ 获得图片 URL 列表
    ↓
组装 HTML 内容
    ↓
创建草稿（draft/add）→ 获得 draft_media_id
    ↓
（可选）发布草稿（freepublish/submit）
```

## 三、详细步骤实现

### 步骤1：获取 AccessToken
**接口**：`GET https://api.weixin.qq.com/cgi-bin/token`

```javascript
const getAccessToken = async (appid, secret) => {
  const url = `https://api.weixin.qq.com/cgi-bin/token?grant_type=client_credential&appid=${appid}&secret=${secret}`;
  const res = await fetch(url).then(r => r.json());
  
  if (res.errcode) {
    throw new Error(`获取 AccessToken 失败: ${res.errcode} ${res.errmsg}`);
  }
  
  return {
    access_token: res.access_token,
    expires_in: res.expires_in // 7200秒（2小时）
  };
};
```

**缓存策略**：建议缓存 token，在过期前刷新（如每1.5小时刷新一次）。

---

### 步骤2：上传封面图（永久素材）
**接口**：`POST https://api.weixin.qq.com/cgi-bin/material/add_material?access_token=TOKEN&type=image`

**封面图规格**：
- **尺寸**：2.35:1（推荐 900×383px）
- **格式**：JPG、PNG、GIF（JPG体积最小，推荐）
- **大小**：≤5MB（建议控制在1MB以内）

```javascript
const uploadCoverImage = async (accessToken, imageBuffer, filename) => {
  const formData = new FormData();
  formData.append('media', new Blob([imageBuffer]), filename);
  
  const url = `https://api.weixin.qq.com/cgi-bin/material/add_material?access_token=${accessToken}&type=image`;
  const res = await fetch(url, {
    method: 'POST',
    body: formData
  }).then(r => r.json());
  
  if (res.errcode) throw new Error(res.errmsg);
  
  return {
    media_id: res.media_id,    // 用于 draft/add 的 thumb_media_id
    url: res.url               // 封面URL（可用于本地记录）
  };
};
```

### 步骤3：上传正文图片
**接口**：`POST https://api.weixin.qq.com/cgi-bin/media/uploadimg?access_token=TOKEN`

**特点**：
- 上传图文消息内的图片，获取微信URL
- **不占用**永久素材库额度
- 仅用于图文消息正文，不可用于封面

```javascript
const uploadContentImage = async (accessToken, imageBuffer, filename) => {
  const formData = new FormData();
  formData.append('media', new Blob([imageBuffer]), filename);
  
  const url = `https://api.weixin.qq.com/cgi-bin/media/uploadimg?access_token=${accessToken}`;
  const res = await fetch(url, {
    method: 'POST',
    body: formData
  }).then(r => r.json());
  
  return res.url; // 返回的图片URL可直接插入HTML
};
```

**批量处理示例**（来自日志）：
```
上传素材：封面 1 个，正文图 24 张；生成草稿 1 个。
```

### 步骤4：创建草稿
**接口**：`POST https://api.weixin.qq.com/cgi-bin/draft/add?access_token=TOKEN`

**关键字段**：

| 字段 | 必填 | 说明 |
|------|------|------|
| `title` | 是 | 文章标题（≤64字节） |
| `thumb_media_id` | 是 | 封面图永久素材ID |
| `content` | 是 | 图文消息正文（HTML） |
| `pic_crop_235_1` | 否 | 2.35:1封面裁剪参数 |
| `pic_crop_1_1` | 否 | 1:1封面裁剪参数 |
| `author` | 否 | 作者（≤8字节） |
| `digest` | 否 | 摘要（≤120字节） |
| `need_open_comment` | 否 | 是否打开评论（0/1） |

#### 封面裁剪参数计算

**场景**：封面图为 2.35:1（900×383px），需适配1:1转发展示

**默认值**：
```json
{
  "pic_crop_235_1": "0_0_1_1",           // 2.35:1 使用全图
  "pic_crop_1_1": "0.287234_0_0.712766_1" // 1:1 居中截取
}
```

**计算逻辑**：
```javascript
function calculateCropParams(originalRatio = 2.35) {
  // 2.35:1 使用全图
  const pic_crop_235_1 = "0_0_1_1";
  
  // 1:1 居中截取
  const cropWidth = 1 / originalRatio;        // 0.4255319...
  const margin = (1 - cropWidth) / 2;         // 0.287234...
  const x1 = margin.toFixed(6);
  const x2 = (1 - margin).toFixed(6);
  const pic_crop_1_1 = `${x1}_0_${x2}_1`;
  
  return { pic_crop_235_1, pic_crop_1_1 };
}
```

**完整请求示例**：

```javascript
const createDraft = async (accessToken, params) => {
  const url = `https://api.weixin.qq.com/cgi-bin/draft/add?access_token=${accessToken}`;
  
  const body = {
    articles: [{
      title: params.title,
      thumb_media_id: params.thumbMediaId,
      author: params.author || "",
      digest: params.digest || params.title,
      content: params.htmlContent,  // 包含<img>标签的HTML
      content_source_url: "",
      need_open_comment: 1,
      only_fans_can_comment: 0,
      pic_crop_235_1: params.picCrop2351 || "0_0_1_1",
      pic_crop_1_1: params.picCrop11 || "0.287234_0_0.712766_1"
    }]
  };
  
  const res = await fetch(url, {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify(body)
  }).then(r => r.json());
  
  if (res.errcode) throw new Error(res.errmsg);
  
  return res.media_id; // 草稿ID
};
```

const buildContent = (imageUrls) => {

## 四、关键注意事项

### 1. 素材删除影响
- **封面图片**：删除素材库中的永久素材，**不影响**已发布文章的封面显示（文章使用独立CDN副本）
- **视频素材**：删除后文章内视频会失效（谨慎操作）
- **清理策略**：可定期删除旧封面素材释放额度（上限10万个）

### 2. 频率限制
- 上传素材有频率限制，建议添加延迟（如每300ms上传一张）
- 遇到 `45009` 错误表示接口调用频繁，需降速

### 3. 图片URL有效期
- `uploadimg` 返回的URL **永久有效**
- 但仅能在微信生态内使用，外部访问可能受限

### 4. 封面裁剪验证
- `pic_crop_235_1` 和 `pic_crop_1_1` 是**可选参数**
- 不传时微信使用默认裁剪（可能非居中）
- 建议始终传入精确计算值

## 五、错误码速查

| 错误码 | 含义 | 解决方案 |
|--------|------|---------|
| `40001` | AccessToken 失效 | 重新获取token |
| `40164` | IP不在白名单 | 添加IP到公众号后台 |
| `41005` | 多媒体文件为空 | 检查文件buffer是否有效 |
| `45009` | 接口调用频繁 | 降低调用频率，添加延时 |
| `40007` | 不合法的媒体文件类型 | 检查图片格式是否为JPG/PNG/GIF |
