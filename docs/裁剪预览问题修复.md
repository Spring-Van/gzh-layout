## 问题分析

裁剪弹窗保存的是**归一化坐标字符串**（如 `0.2_0.1_0.8_0.9`），但封面预览区域用的是 `backgroundPosition: center`，完全没有读取裁剪坐标来显示对应区域。

需要将裁剪坐标转换为 CSS `background-position` + `background-size` 来实现预览。

---

## 解决方案

### 第一步：新增工具函数 `cropToBackgroundStyle`

```typescript
/**
 * 将归一化裁剪字符串转换为 CSS background 样式
 * cropStr 格式: "x1_y1_x2_y2"，值为 0~1 的归一化比例
 */
function cropToBackgroundStyle(
  imageSrc: string,
  cropStr: string | undefined,
): Record<string, string> {
  if (!imageSrc) return {};

  // 无裁剪数据时，默认铺满
  if (!cropStr) {
    return {
      backgroundImage: `url(${imageSrc})`,
      backgroundSize: "cover",
      backgroundPosition: "center",
    };
  }

  const parts = cropStr.split("_").map(Number);
  if (parts.length !== 4 || parts.some(isNaN)) {
    return {
      backgroundImage: `url(${imageSrc})`,
      backgroundSize: "cover",
      backgroundPosition: "center",
    };
  }

  const [x1, y1, x2, y2] = parts;
  const cropW = x2 - x1; // 裁剪区域宽度比例
  const cropH = y2 - y1; // 裁剪区域高度比例

  if (cropW <= 0 || cropH <= 0) {
    return {
      backgroundImage: `url(${imageSrc})`,
      backgroundSize: "cover",
      backgroundPosition: "center",
    };
  }

  // background-size: 让整张图缩放后，裁剪区域恰好填满容器
  // 容器宽度 = cropW * 图片宽度 => 图片缩放后宽度 = 100% / cropW
  const bgSizeW = (1 / cropW) * 100;
  const bgSizeH = (1 / cropH) * 100;

  // background-position: 裁剪起点对应容器左上角
  // position% 的含义：(图片宽 - 容器宽) * position% = offset
  // offset = x1 * 图片原始宽 * scale = x1 / cropW * 容器宽
  // => position% = (x1 / cropW) / (1/cropW - 1) = x1 / (1 - cropW)
  const bgPosX = cropW >= 1 ? 0 : (x1 / (1 - cropW)) * 100;
  const bgPosY = cropH >= 1 ? 0 : (y1 / (1 - cropH)) * 100;

  return {
    backgroundImage: `url(${imageSrc})`,
    backgroundSize: `${bgSizeW}% ${bgSizeH}%`,
    backgroundPosition: `${bgPosX}% ${bgPosY}%`,
  };
}
```

### 第二步：修改模板中封面预览的 `:style` 绑定

**2.35:1 大图预览区域**（`selectedCoverRatio === '235'` 分支）：

```vue

<div
  class="w-full h-full"
  :style="{
    backgroundImage: `url(${generatedCoverImageSrc})`,
    backgroundSize: 'cover',
    backgroundPosition: 'center',

  class="w-full h-full"
  :style="
```

## 原理说明

| CSS 属性              | 计算逻辑                                                      |
| --------------------- | ------------------------------------------------------------- |
| `background-size`     | `100%/cropW` × `100%/cropH`，让图片放大使裁剪区域恰好填满容器 |
| `background-position` | `x1/(1-cropW)*100%`，将裁剪起点对齐容器左上角                 |
